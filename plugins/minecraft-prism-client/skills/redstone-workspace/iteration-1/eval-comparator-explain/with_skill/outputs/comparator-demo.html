Comparator Subtract Mode Demo

A simple test rig to see subtract mode in action: rear input minus side input controls a redstone lamp

Dimensions 7 x 3 x 2
Layers 2
Total Blocks 15
Difficulty Easy

Materials Needed

How It Works

The comparator has three inputs: a rear input (the main signal coming in from behind), and two side inputs (from the left and right). It outputs from the front.

In subtract mode (right-click the comparator once so the front torch lights up), the comparator takes the rear signal strength and subtracts the strongest side signal from it. The formula is: output = rear - max(left side, right side). If the result would be negative, it outputs 0 instead.

In this demo build: The lever on the left provides a rear input at signal strength 15. The repeater on the right side feeds a side input, also at strength 15. With the side lever OFF, the comparator outputs 15 - 0 = 15, and the lamp lights up. With the side lever ON, the comparator outputs 15 - 15 = 0, and the lamp turns off.

Try it yourself: Build this in creative mode. Place both levers. Turn on the rear (left) lever first -- the lamp lights. Then turn on the side (bottom) lever -- the lamp goes dark. The side signal is being subtracted from the rear signal. You can also experiment by extending the side dust path (making it longer reduces signal strength), and watch how the lamp gets dimmer as the subtraction amount changes.

Why subtract mode is useful: It lets you do math with redstone signals. You can use it to detect "how full is this chest?" by comparing container signal levels, create analog circuits that respond to specific signal thresholds, or build counters and combination locks.
